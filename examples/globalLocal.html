Dinge Bewegen

In einer einefachen Zweidimensionalen Welt werden Dingen, wie eine Ellipse, in einem Programmfenster positioniert in dem man ihnen Werte auf der y- und x-Achse zuschreibt. Um so eine Ellipse in Bwegung zu bringen muss man wenigsten einen dieser Werte verändern. Kontinuierliche Veränderung sorgt dann für kontinuierliche Bewegung. Ein gesetzter Anfangswert x = 20; kann mit jedem Durchlauf des Draw-Loops um einen gewünschten Wert erhöht werden.


Damit x nicht mit jedem Durchlauf des Draw-Loops auf den Ausgangswert (20) zurück gesetzt wird, muss diese Initialisierung außerhalb erfolgen.

Variablen die innerhalb einer Funktion initialisiert werden, stehen nur lokal in dieser Funktion zur Verfügung. Sie sind für den Rest des Codes nicht nicht zugänglich. Damit die Variablen x und y in der Draw-Funktion verwendet verwendet werden können. müssen sie vor der Setup-Funktion initiert werden. Damit stehen sie global für alle Teile des Programms zur verfügung.

x = 20;
y = 40;
ellipse(x,y,100,100);
x = x +2.4;

Variablen Initieren

Mit dem Zauberwort float() vom englischen Floating Point Numbers werden Kommazahlen initiert. Einer Variable kann direkt beim initieren ein Wert zugewiesen werden (siehe x). Dies kann aber auch nachgeholt werden (siehe y).

Bevor eine Variable benutzen werden kann muss sie mit dem Zauberwort für den entsprechenden Datentyp initiert werden. Für Kommazahlen engl. Floating Point Numbers ist das Zauberwort float. Der Datentyp für ganze Zahlen int von englisch Integer kann keine Kommazahlen enthalten. Man kann einer Integer einmal 0.5 hinzufügen.

Einfache Datentypen

booleanEinfach Entweder / Oder-Werte
StringZeichenketten: Worte, Sätze aber auch Dateipfade
charEinzelne Zeichen, Buchstaben, Zahlem

Lokale Variablen

Werden Variablen innerhalb von einer Funktion initiert, können die auch nur innerhalb dieser Funktion aufgerufen werden. Man spricht von Lokalne Variablen. Außerhalb der Funktion bleiben diese unbekannt. Dies gilt auch für die Funktionen setup() und draw().

Im Loop kleben bleiben

Möchte man eine Ellipse über die Veränderung der x-Position im Draw-Loop animieren, kann man sie nicht dort innitieren. Sonst geschiet bei jedem Durchlauf folgendes:
Die Ellipse startet von x = 4
Sie wird anschließend um 2.2 nach rechts verschoben
Der Loop startet von neu. Die Ellipse wird wieder auf 4 zurückgesetzt.
Die Funktion println() gibt das Gezappel in der Konsole aus.

Auch wenn x nur im Draw-Loop benutzt wird, kann er hier nicht Initiert werden. Sonst bleiben wir kleben. 

void  setup(){
    size(640,360); 
}
void draw(){
    float x = 4;
    float y = height / 2;
    float xSpeed = 2.2; 
    ellipse(x,y,100,100);
    x = x + xSpeed;
    println(x);  
}

Los geht's

Wird eine Variable vor der Setup-Funktion initiert, ist diese global verfügbar und kann von allen Teilen des Programms benutzt werden. Einer Variable kann direkt beim initieren ein Wert zugewiesen werden (siehe x). Dies kann aber auch nachgeholt werden (siehe y).

Nachdem innerhalb von setup() mit size() oder der Funktion fullScreen() die Größe des Programmfensters festgelegt wurde, stehen daraufhin Werte für Fensterbreite (width) und Fensterhöhe (height) zur Verfügung.
y kann nun der Wert für halbe Fensterhöhe (height / 2) zugewiesen werden.


innerhalb von Draw sorgt die Funktion background() dafür, dass der Hintergrund mit jedem Frame neu gezeichnet wird und alte Positionierungen somit verschwinden.

Die Ellipse wird auf die neuste x-Position gezeichnet. Mit jedem Frame wird x um den Wert von xSpeed erhöht, bis die in if() formulierte Umkehrbedingung erfüllt ist.

Sobald x den beschriebenen Grenzwert (Programmfenster-Breite + Ellipsen-Radius) erreicht und damit rechts aus dem Bild verschwindet, wird x zurück gesetzt auf 'Minus Ellipsenradius' und taucht dann von Links wieder ins Bild ein.


Die Funktion println() ist ein Hilfsmittel zur Fehlersuche. Sie gibt gewünschte Were an die Konsole unterhalb des Code-Fensters in der Processing App aus.

float x = 4;
float y;

void  setup(){
    size(640,360); 
    y = height / 2;
}

void draw(){
    float xSpeed = 0.2; // Geschwindigkeit
    int r = 50; // Radius
    background(0,30,50);
    noStroke();
    fill(255,200,50);
    ellipse(x,y,r*2,r*2);
    x = x + xSpeed;
    if(x >= width  +r){
        x = -r;
    }
    println(x);
}
